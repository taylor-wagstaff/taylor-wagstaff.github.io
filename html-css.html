What is the difference between Margin, Border, and Padding?





margin

/ˈmɑːdʒɪn/



noun

1.

The edge or border of something.



border

/ˈbɔːdə/



noun

1.

The edge or boundary of something, or the part near it.



padding

/ˈpadɪŋ/



noun

1.

Soft material such as foam or cloth used to pad or stuff something.





Following each of these definitions above we can begin to understand what these terms mean individually within a CSS context. If we follow the CSS box model, which simply means a box or container holding content, such as text, we can visualize how each of these terms signify how they are meant to be understood. Each box can contain margins, borders and padding.

Above, we can see how each of these terms relate to their inferred meaning. The margin, by the definition above, functions as the edge occupying around the content of the box. Think of the margin as the grass that surrounds the castle.


Second is the border, which is nested between the margin and padding. The border is the boundary or the castle’s outer walls so to speak.


Finally the padding, if we return to the above definition as a “soft material such as foam or cloth used to pad or stuff something” is a helpful way to visualize how the padding operates. The padding, which is normally the same as the background color as defined for the box container, packs the content or wraps the content in an area of empty space.


When applied in CSS, the margin, padding and border can all be increased or decreased in size through declaring how much or how little in number declarations.




We can then see each specific properties for the margin, padding and border. In the above example in the browser, the border is set as ‘solid thin black’, the margin has a size of ‘2em’ and the padding is ‘2em’ as well. An ‘em’ is a unit of measurement, although ‘px’ for pixels can also be used.

You can also specify margin, padding and border for each side of the box through:


The CSS code is written like this:

  • margin-top:
  • margin-right:
  • margin-bottom:
  • margin-left:

Or as one piece of code to represent all four sides:

You can also specify many different border styles such as:



For example, to represent a ‘dashed border’ you would write:


border: dotted;


To specify size of the border with one value:


border-width: 1px;


When to use Margins, Padding and Borders:


In regular use margins are good for adjusting the spacing of an element in conjunction with another element, i.e. positioning elements in relation with each other. Padding is good for when you're adjusting the look of a particular element like changing the space or amount of pixels contained within the box or div. Borders are useful to separate elements from other elements or containers. They are also handy when you're arranging elements on a page to clearly see where everything is located.
